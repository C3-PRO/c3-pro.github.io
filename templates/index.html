What is C3-PRO?

C3-PRO is a set of FHIR® compliant software tools and systems that enables integration of biomedical research data collected via ResearchKit and ResearchStack into i2b2 and other databases in a secure, highly scalable fashion.

For Developers

Source code for all components is available on GitHub. We've set up a Google Group for discussions and updates, feel free to stop by and ask questions.

Modules!

The ecosystem supporting mobile research is expanding, and the list of modules that plug in to C3-PRO is growing. See the architecture overview below to find software modules that suit your needs.

Open Source, Open Access

All components of C3-PRO are available open source, Apache 2 licensed.
