What is C3-PRO?

C3-PRO is a set of FHIR DSTU-2 compliant software systems that enables the integration of Apple's Research Kit data into i2b2 in a secured and highly scalable fashion. Read more...

Used in real studies

C Tracker is the first research study utilizing C3-PRO. The study evaluates the impact of hepatitis C on people’s lives by enabling patients to participate from their iPhones.

i2b2 Integration

C3-PRO stores the gathered data into an i2b2 instance, the widely adopted scalable data mining system for clinical research.

ResearchKit

C3-PRO uses Apple's ResearchKit framework to consent patients and to drive surveys and activity reports directly on patients' iPhones.

FHIR DSTU-2 compliant

C3-PRO subsystems use FHIR DSTU-2 standards. From iPhone to i2b2, data is transmitted and captured using FHIR DSTU-2 resources.

Highly scalable and secure

Data is encrypted on the iPhone and temporally stored in highly reliable AWS instances. Decryption happens in-house using RSA and AES encryption algorithms.
