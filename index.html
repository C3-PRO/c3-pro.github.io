Bridging the gap between ResearchKit and i2b2 with FHIR

What is C3-PRO?

C3-PRO is a set of FHIR compliant software systems that enables integration of data collected via ResearchKit into i2b2 in a secure, highly scalable fashion.

Leading Technologies, Combined

i2b2 Integration

C3-PRO stores gathered data into an institutionally hosted i2b2 instance, the widely adopted scalable data mining system for clinical research.

ResearchKit

C3-PRO uses Apple's ResearchKit framework to consent patients and to drive surveys and activity reports directly on patients' iPhones.

FHIR compliant

C3-PRO subsystems use the FHIR üî• standard: from iPhone to i2b2, data is captured and transmitted using FHIR resources.

Highly scalable and secure

Data is encrypted on the iPhone and temporally stored in highly reliable AWS instances. The RSA and AES cryptography algorithms are used for encryption.
