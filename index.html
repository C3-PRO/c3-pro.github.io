What is C3-PRO?

C3-PRO is a set of FHIR¬Æ compliant software tools and systems that enables integration of biomedical research data collected via ResearchKit and ResearchStack into i2b2 and other databases in a secure, highly scalable fashion.

For Developers

Source code for all our modules is available on GitHub. We've set up a Google Group for discussions and updates, feel free to stop by and ask questions.

Architecture
